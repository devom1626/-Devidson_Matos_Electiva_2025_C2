# "simplificador de uso de git" # 

errores al subir un git:

1 quieres subir una carpeta que contiene otra carpeta que es un repositorio.
una solucion seria eliminar dicha carpeta del bash para resubir la nueva donde esta estara incluida.
usa: "rm -rf .git"  
abras eliminado todas las carpetas y repositorios cargados a tu git bash y estara limpio para tu nuevo init, si vas a "git status" todo estara limpio.

borrar carpeta como repositorio principal: 

2 si no quieres borrar todo y solo quieres que una carpeta deje de ser repositorio usa:
"rm -rf carpetaX/.git"



 archivo ya en remoto: 

3  Si quieres borrar del git hub.
SI QUIERES BORRAR ALGO DE GH PERO DESEAS MANTENERLO EN EL BASH SOLO INTRODUCE EL COMANDO:
 "git rm -r --cached ."
 "git commit -m "Elimine  los archivos del remoto"
 "git push origin main/master" ojo elige main o master segun hayas usado.

 borrar carpeta especifica:

4 si quieres borrar una carpeta en especifico de gith bash y GitHub dentro de un repositorio puedes usar: 
"git rm -r NIVARAX"
"git commit -m "Eliminar carpeta NIVARAX"
"git push"
